Python Variables

Variables are a way you can store values when you want to use them later. You can create a variable like this: 
<variable-name> = <variable-value>
Replace 
<variable-name>
and 
<variable-value>
with the name of the variable and the value of the variable.
Throughout this course, there will be pieces of code that you can run. Click the button on this to run it:

The name of the variable can only contain letters, $, _, and numbers. It also cannot start with a number. The value of a variable has to be a valid Python data type (covered in the next topic).

Python Comments

Sometimes, Python programs can get very big. It can get confusing about what a piece of code does. Comments can help with that. You can insert a comment that explains what a piece of code does. A comment starts with a "#". You can put anything inside a comment. Python ignores comments. 
myVariable = 999 # This comment will be ignored
or 
myVariable = 999
# This comment will be ignored
